Locadora de Veículos Online

É uma plataforma versátil e eficiente projetada para facilitar o gerenciamento e a interação entre usuários, funcionários e locadoras de veículos. A plataforma pode ser utilizada por empresas de aluguel de carros para adcionar seus funcionarios e veículos, tornando possivel gerenciar suas diversas filiais e estoques de veículos através do sistema, e tambem por clientes para buscar veículos disponíveis em uma determinada locadora, fornecendo uma visão abrangente das opções disponíveis para atender às suas necessidades de locomoção. A Locadora de Veículos Online oferece uma solução completa para atender às necessidades de locação de veículos, proporcionando uma experiência fluida tanto para os usuários quanto para as locadoras.

Este projeto foi desenvolvido por Leila Hortencio, aluna do curso de Engenharia de Software no Instituto Infnet. Ele serve como um teste de proficiência na disciplina de Desenvolvimento de Serviços Web e Testes com Java, ministrada pelo professor Elbert Moraes. O projeto representa um esforço dedicado em compreender e aplicar os princípios da linguagem Java desenvolvendo um serviço web, bem como explorar as capacidades de configuração do ambiente de desenvolvimento, criação um sistema funcional, manipulção de dados e consulmo de APIs externas utilizando o framework Spark Java.
